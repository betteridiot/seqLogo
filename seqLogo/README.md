seqLogo
===============================

version number: 0.0.1
author: Marcus D Sherman

Overview
--------

Python port of the R Bioconductor `seqLogo` package 

Installation / Usage
--------------------

To install use pip:

    $ pip install seqLogo


Or clone the repo:

    $ git clone https://github.com/betteridiot/seqLogo.git
    $ python setup.py install
